# Global Codex preferences

When a task involves app UI, frontend screens, UX, design systems, dashboards, SaaS interfaces, mobile layouts, onboarding, navigation, forms, landing pages, or visual polish, use the `premium-product-ui` skill.

Default UI quality bar:
- product-first, not decoration-first
- clean, premium, modern, usable
- strong hierarchy and spacing
- reusable components
- responsive and accessible states
- no generic template look
