# MVP scope

Prva lokalna MVP verzija služi za provjeru osnovnog viralnog loopa: napravi leksikon, podijeli link u WhatsApp/Viber, prijateljica se upiše, vlasnica čita upise preko privatnog linka.

## Ulazi u MVP

1. Landing stranica.
2. Kreiranje leksikona.
3. Javna pozivnica leksikona.
4. Forma za upis prijateljice.
5. Thank-you stranica nakon upisa.
6. Privatni admin pregled putem tokeniziranog linka.
7. Edit/delete link za osobu koja se upisala.
8. Privacy i Terms stranice.
9. Open Graph preview kasnije.
10. Prvi Convex backend slice za leksikone, upise i privatne tokene.

## Ne ulazi u MVP

- Login ili registracija.
- Korisnički profili.
- Javni feed.
- Javno pretraživanje leksikona.
- Komentari.
- Upload slika.
- AI generiranje slika.
- Plaćanje.
- Premium teme.
- PDF export.
- Email slanje.
- Push notifikacije.
- Kompleksna admin analitika.
- Google/Meta tracking.
- Cookie consent banner dok nema neesencijalnih kolačića.

## Idealni user flow

1. Korisnica otvara landing.
2. Kreira leksikon.
3. Dobiva javni link i admin link.
4. Šalje javni link u WhatsApp/Viber.
5. Prijateljica se upisuje.
6. Nakon upisa vidi CTA da napravi svoj leksikon.
7. Vlasnica gleda upise u privatnom adminu.

## Granice trenutnog koraka

Trenutni backend slice pokriva kreiranje leksikona, javnu pozivnicu, formu za upis prijateljice, privatni admin pregled i edit/delete flow za pojedini upis. Još nema kviza, Open Graph previewa, uploada, autha, plaćanja, javnog feeda ili vanjskog trackinga.
